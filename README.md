 Smart Refrigerator System (C)

 Description: 
This project is a console-based smart refrigerator system developed in C.  
It allows users to manage food materials, track expiration dates, and get recipe suggestions.

Features:
- Add / remove materials
- Track expiration dates
- Low stock warning system
- Recipe suggestion based on available ingredients
- Sorting materials alphabetically

Purpose:
This project was developed during my first year to practice:
- Arrays and string manipulation
- Functions and modular programming
- Time and date operations in C

Technologies:
- C Programming Language
- Standard Libraries: stdio.h, string.h, time.h

What I Learned:
- Managing structured data without using structs
- Working with date and time in C
- Building a multi-functional menu-driven system
- Problem solving and feature design

Known Limitations:
- Uses global arrays instead of dynamic structures
- Date validation is basic
- Platform-dependent functions (e.g., strcasecmp)

Future Improvements:
- Refactor using structs
- Add file storage (persistent data)
- Improve date validation
